How it works

Six steps, fully automated, end-to-end.

🎬
01

Upload Video

Drop in any short educational video — lecture, tutorial, explainer.

🔊
02

Audio Extraction

FFmpeg strips the audio track as a clean 16 kHz WAV for processing.

📝
03

Transcription

Whisper Large V3 via Pollinations API transcribes speech with precise timestamps.

🌍
04

Translation

GPT-4o via Pollinations translates each subtitle segment into the target language.

🗣️
05

Voice Cloning + TTS

Chatterbox clones the speaker's voice and synthesises the translated speech.

⏱️
06

Sync & Export

Each audio segment is stretched or padded to match the original timing, then merged back into the video.
